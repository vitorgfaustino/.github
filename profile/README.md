# E aí, Vitor aqui



Não sou desenvolvedor de carreira. Sou empresário, gestor de tráfego e entusiasta de tecnologia.
Gosto de testar stacks, quebrar coisas, consertar depois e transformar ideias em produtos reais.

Tenho foco em aplicações rápidas, simples, privadas, com baixa dependência e pouco espaço para vendor lock-in.
Software foi feito para servir, não para prender. Se a ferramenta sequestra dados, complica o simples ou força dependência de uma única plataforma, para mim é tchau e bênção.

- **CEO na [Alpes Data](https://alpesdata.com.br)**: estratégia, dados, tráfego pago, automação e negócios digitais.

## Minha stack de sobrevivência digital

- **Web e performance:** Go, HTMX, SQLite, Hugo, Cloudflare Pages, Cloudflare Workers e aplicações serverless.
- **Infra e autonomia:** Linux, Docker, Git, SSH, redes, automação e self-hosting.
- **Filosofia:** simples antes de complexo, privado antes de conveniente, portátil antes de dependente.

Se o sistema precisa de um bilhão de dependências para fazer o básico, já começou errado.
E se não for Hot Swap, não serve.
